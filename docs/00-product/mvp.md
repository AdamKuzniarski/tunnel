# MVP

## Main promise

Help the user start to protect a focus session with more friction than a normal timer app.

## MVP Scope 
- onboarding and permission flow
- choose distracting apps
- start on focus session
- basic shielding / blocking concept on iOS
- session history at a basic level

## Possible later additions

- NFC-based unlock ritual

## Out of scope for MVP

- Android feature parity
- polished analytics dashboard
- social features
- AI coach
- custom hardware
- complex backend
- web app as core product surface
