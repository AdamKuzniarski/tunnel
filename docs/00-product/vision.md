# Vision

tunnel is an iPhone-first focus app.

The core is simple:
a user starts a focus session, selected apps become harder o impossible to access, and unlocking should feel intentional rather than casual.

This project is based on one main belief:
small amounts of friction can change behavior.

## Core concept

- the user select distractcing apps
- a focus session can be started
- access becomes restricted during the session
- unlocking should feel require intention and friction 
- later, NFC may become part of that unlock ritual

## Product direction

tunnel is not meant to be a social network, a gamified productivity toy, or a gigant platform from day one.

The first direction is:
- presonal focus
- strong simplicity
- iPhone-first exectution
- real platform-aware constraints
